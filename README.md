### 👨‍💻 About Me

I'm a Game Developer since 2022, working with **Unreal Engine (C++)** and **Unity (C#)**.

**Currently:** Lead Unreal Engine Developer @ [IMT Games](https://imt.games/)  
**Working on:** Mobile Battle Royale (Android) — [Google Play](https://play.google.com/store/apps/details?id=com.imt.worldfightarena)  
**Previously:** DEVIL GATE (co-op Action/RPG for Steam) — [Steam Page](https://store.steampowered.com/app/4553510/DEVIL_GATE/)

I love building solid game mechanics, working with complex systems, and shipping games.

> 🎯 **Currently exploring:** Unity & C# — network solutions (Photon/Mirror) and URP.
